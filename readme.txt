A custom maintenance mode plugin by Artificial Heights, LLC. Includes a toggle to enable maintenance and supports fully custom HTML blocks.\
\
== Description ==\
\
**Artificial Maintenance** lets you display a sleek, branded maintenance screen with your own HTML, animations, or design. Built for speed and flexibility, it\'92s ideal for agencies, startups, and tech innovators.\
\
Features include:\
- Toggle to enable/disable maintenance mode\
- Custom HTML block editor\
- Responsive output\
- WordPress-compatible update delivery\
\
== Installation ==\
\
1. Upload the plugin files to the `/wp-content/plugins/artificial-maintenance` directory, or install directly from GitHub.\
2. Activate the plugin through the 'Plugins' screen in WordPress.\
3. Navigate to 'Artificial Maintenance' under the admin menu to configure.\
